# Font-Bot

A simple python oriented telegram bot to give out creative font style's

## REQUIREMENTS

tgcrypto<br>
pyrogram==1.2.9

## Installation

Fork this repository and
Put the Neccesory Tokens Here when Deploy<br>
     ` BOT_TOKEN`<br>
      `API_ID`<br>
      `API_HASH`<br>
      `OWNER_ID`
